O Provedor de ícones padrão é o Material Icons, que é o utilizado pelo time de UX nos protótipos.

Provedores válidos: `mi, fa, el` (Material Icons, Font Awesome, Element UI Icons)

A prop `type` indica o tipo de ícone que será renderizado. Padrão é `outlined`

Exemplos: Material Icons: `filled, outlined, etc.`; Font Awesome: `fa, fas, far`;

Estes ícones são renderizados como texto, por isso, basta estiliza-los como tal no css.
